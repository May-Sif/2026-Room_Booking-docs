# 2026 Room Booking System - Documentation

## Project Overview
Room Booking System is a web-based application developed to manage campus room reservations in a centralized and structured manner.

The system solves problems such as:
- Scattered booking records
- Difficulty monitoring booking status
- No centralized history tracking
- Lack of structured evaluation data

## Project Components

| Component | Technology |
|------------|------------|
| Backend | ASP.NET Web API |
| Frontend | React + TypeScript |
| Database | SQLite|
| Version Control | Git + GitHub |
